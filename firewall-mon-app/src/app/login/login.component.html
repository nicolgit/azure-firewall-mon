az-firewall-mon

Welcome to Az-firewall-mon: a tool that allows to view, search, filter and inspect the log stream of your Azure Firewall. Connect your Azure Firewall to this tool and start monitoring your network traffic (how?)

demo mode
Event Hub connection string
Event Hub Consumer Group
Azure Maps Account Shared Key (optional)

The following optional parameters are needed if you want to use natural language to filter the log stream.

Azure OpenAI endpoint
Azure OpenAI deployment for a gpt-4-32k v613 model
Azure OpenAI access key
